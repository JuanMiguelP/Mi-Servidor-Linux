# Ejercicio de Gestión de Servicios
Este proyecto demuestra cómo controlar servicios del sistema en Ubuntu Server.

## Descripción:
- **Tarea**: Instalar, comprobar el estado y detener el servicio Apache2.
- **Gestión de recursos**: Verificación del uso de memoria RAM al detener servicios que no se usan.

## Comandos utilizados:
- `systemctl status`: Para comprobar el estado (si está encendido o apagado).
- `systemctl stop`: Para detener el servicio y liberar recursos.
